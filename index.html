Commands

Prefix Is !! 

  • !!play |Play a song with the given name or url
  • !!stfu |Disconnect the bot from the voice channel if is in
  • !!pause |Pause the currently music playing
  • !!resume |Resume paused music
  • !!skip |Skip the currently playing song
  • !!vol |Change the current volume
  • !!np |Shows what song the bot is currently playing
  • !!queue |View the queue

    • ––––•(-• Other Commands •-)•–––– 

  • !!avatar |Gets a user's avatar
  • !!botinfo |Gets a bot info
  • !!userinfo |Gets a user info
  • !!serverinfo |Gets a server info/stats
  • !!invite |Generates a link to invite LMEJDOUB-BOT to your server

    • ––––•(-• Ticket Commands •-)•–––– 

  • !!new |Open a new ticket and tags the Support Team
  • !!close |Close a ticket
